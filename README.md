# coffee-churn-prediction
Прогнозирование оттока клиентов сети кофеен на основе логистической регрессии

# Основные результаты
Достигнут PR AUC = 0.7163 на тестовой выборке.
Установлена критическая связь между технической стабильностью приложения и оттоком (корреляция сбоев с уходом клиента — 0.85).
Модель в 11 раз эффективнее случайного классификатора.

# Технологический стек
Язык: Python
Библиотеки: Pandas, NumPy, Scikit-learn, Matplotlib, Seaborn
Алгоритм: Logistic Regression с L1-регуляризацией (Lasso)

# Структура репозитория
coffee_churn.ipynb — основной файл с исследованием, визуализацией (EDA) и построением модели.
coffee_churn_model.joblib — сохраненный финальный пайплайн (препроцессор + модель).
requirements.txt — список зависимостей для воспроизведения среды.

# Особенности данных
Данные для проекта предоставлены учебной платформой. Доступ к исходному датасету ограничен внутренним хранилищем сервиса. В коде тетрадки зафиксированы все этапы предобработки, генерации признаков (логарифмирование, бинаризация) и финальные выводы.

# Рекомендации для бизнеса
Первоочередной мерой по удержанию клиентов является устранение технических ошибок в мобильном приложении. Даже лояльные клиенты с Pro-подпиской склонны к оттоку после 3-4 сбоев в месяц.
